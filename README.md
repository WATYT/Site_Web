Bonjour,

Dans le cadre du module de WEB en LP MEEDD, nous avons du réaliser un site web de notre choix.
J'ai choisi de réaliser une version illégale du site connu, leboncoin.
Sur le site, vous avez accès à la création de compte, l'ajout d'annonce, la consultation d'annonce, la suppression d'annonce ainsi qu'à un tchat pour discuter entre les utilisateurs.

Pour ouvrir le projet, il vous faudra WAMP64.
Vous devrez remplacer le fichier www du logiciel par celui du projet.
Il faudra ensuite ouvrir phpmyadmin et importer la base de donnée.
Bonne chance à vous.
